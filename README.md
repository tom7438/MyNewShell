MyNewShell
Groupe 11

Compilation :
Shell V1 : make
Shell V2 : make MAIN2=1
Shell V3 : make MAIN3=1

Option de compilation :
DEBUG : affiche les messages de debug
MAIN2 : compile la version 2
MAIN3 : compile la version 3
TEST : enlève l'affichage du prompt

Exécution :
Shell V1 : ./Bin/shell1
Shell V2 : ./Bin/shell2
Shell V3 : ./Bin/shell3
